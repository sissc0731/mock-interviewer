## 🚀 快速开始

```bash
git clone https://github.com/sissc0731/mock-interviewer.git
cp -r mock-interviewer/.claude ./
```

然后说：

> "模拟一次前端面试，3年经验，React方向"

## 🎭 面试流程

```
自我介绍（30秒）
    ↓
技术基础（3-5题，由浅入深）
    ↓
项目深挖（根据你的回答追问）
    ↓
算法题（1-2道，现场写代码）
    ↓
反问环节（你问面试官）
    ↓
📊 面试评估报告
```

---

## 💡 特色

- 🔁 **实时追问**：根据你的回答深度追问，不是死板题库
- 📊 **评估报告**：每轮结束给出评分+改进建议
- 🎯 **定级模拟**：模拟阿里P6-P8、字节2-1~3-1等不同职级
- 😰 **压力面模式**：开启后面试官会质疑你的每个回答
